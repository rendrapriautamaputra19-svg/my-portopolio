# My Portopolio (Simple Landing Page)

Sebuah project website portofolio pribadi yang dibuat sesederhana mungkin. Fokus utama dari project ini bukan pada tampilan yang ramai, melainkan pada efisiensi fungsi di latar belakang.

## 🛠️ Tech Stack yang Digunakan

- **Next.js & Tailwind CSS:** Digunakan untuk menyusun struktur halaman dan layout yang simpel agar performa web tetap ringan.
- **Form System:** Halaman landing page dilengkapi form input minimalis yang sudah terintegrasi dengan sistem automasi data di _background_.

## 📌 Catatan Project

Project ini adalah eksperimen pribadi untuk membuktikan bahwa tampilan yang sederhana pun bisa bekerja secara cerdas dan otomatis di belakang layar.
